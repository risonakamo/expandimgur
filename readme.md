## new key controls
<kbd>ctrl</kbd>+<kbd>shift</kbd>+<kbd>s</kbd>: sort by date descending

<kbd>ctrl</kbd>+<kbd>shift</kbd>+<kbd>a</kbd>: open imgur album page for the currently selected album
